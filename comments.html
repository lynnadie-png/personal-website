Structure of Lynn's website

The pages are structured from top to down; title, navigation bar, headings, body information followed by the footer at the bottom of each of the pages.

What the pages are & how they interact

The pages of my website are The Home Page, My Past, My Future, and My Comments pages. These pages interact through the navigation bar, which lists the four different pages.

The Home Page

  • Home Page Structure

    This allows the page of the Home Page to be viewed in the order; tab header, title, navigation bar, subheading, body paragraph, then the footer.

          Styles used in the website:

          Tag Selectors

          Selector used What the Style does Where the Style is used
          body It changes the background colour to the cream/ off white colour It is used for all the pages, changing the background colour of all the pages on the website to off-white
          ol (Ordered list) This ordered list is used for the navigation bar This is used throughout all the website pages, allowing for the viewer to click on any of the four pages - Home, Past, Future and Comments page.
          li (list) This focuses on the aesthetic presentation of the navigation bar; font, font colour and size, background colour, hover colour, size of the navigation bar, positioning, border. This is used throughout all of the four pages of the website, allowing the navigation bar and options to look the same across the website.
          body This allows the font type to be Courier This is used throughout the entire website, allowing the font of the headings, subheadings and body to be the same.
          h1 (Header 1) This allows for Header 1 to be underlined. This is used across all four pages, underlining all the headings of the pages
          h2 (Header 2) This allows for Header 2 to be underlined and controls the colour and thickness of the underline. This is used for the Home and Comments pages, underlining the headings of the pages
          p This controls the size, font and colour of text in the document paragraphs. This was used in all Comments, Home, Past and Future pages text.
          body This ensures that the body covers the entire web page This is used across all the website pages.
          main This searches the website page to ensure that the footer doesn't float and remains at the bottom of the web page This is used across all the pages, ensuring that the footer stays at the bottom of the pages.
          footer This organises the footer appearance; colour of the box and text, size and positioning. This was used for all the web pages, they all have a footer.


          Class Selectors

          Selector used What the Style does Where the Style is used
          .nickname This allows for the nickname to be personalised aesthetically This is used for the Home page, putting writing in italic, bold and in a mauve colour
          .main-card This allows for a text box to appear and be personalised, creating a border and background colour This is used for all the pages to create a white text box and mauve background
          .image-caption This creates a caption for images, controlling font, position, colour and size This is used for the Past and Future Pages, creating a caption beside the images
          .img-left This controls the size, border and position of the left image This is used for the Past Page and Future Page, creating a suitable position for the images
          .img-right This controls the size, border and position of the right image This is used for the Past Page and Future Page, creating a suitable position for the images
          .text-wrap-container This allows the text to wrap around the images This is used for the Past Page and Future Page, allowing the text to be be around the images.
          .container This allows the text to be centered This is used across all the website pages, allowing the texts to be centered within the border of the pages.
          .tree This created the tree for the information to be organised, and also controls its colour and appearance. This was used in the Comments Page, to organise the structure of the Home, Past, Future and Comments pages.
          .circle-list This creates a standard bullet list for the list items and creates padding so the items don't hit the edge of the page. This was used for lists in Graphic design aspects of the website
          .my-table This creates a table, and controls the colour, size, position and border. This was used for the Comments page, for this table


          ID Selectors

          Selector used What the Style does Where the Style is used
          #graphic-design-header This uniquely underlines and controls the colour of the subheadings Used for the heading for graphic design elements in the Comments page.

          Graphic design aspects of the website

          Layout

          • Navigation bar - increases usability
          • Headers and subheaders - larger headers and smaller subheadings adds visual hierarchy
          • Alignment - content is aligned to the left
          • CSS flexbox - ensures that the footer remains at the bottom of the web pages
          • Text box - ensures that the content doesn't touch the edges of the page, allows it to look aesthetically pleasing

          Colours

          • Background Colour - a cream colour which creates contrast with the black text
          • Navigation Bar & Footer - both a deep mauve colour
          • Hover Colour - a light pink when hovering over options on navigation bar

          Typefaces

          • Font of the navigation bar options is in capitals and sans-serif
          • The Headings are underlined, emphasising the page titles
          • Font of headers, body and footer is in Courier, providing a uniform look
          • Font of headings and subheadings are in bold

          Visual Elements

          • Images - provide a connection between the viewer and my past
          • Tables - provides an aesthetically pleasing way of presenting and organising information without overcrowding

          How the (alt) tag is used

          The (alt) tag is used as an attribute inside the (img) tag, and stands for alternative text. If for some reason, the image does not work on the website, the (alt) tag shows text which appears instead of the image. I have used the (alt) tag for my images in My Past and My Future pages.

          How the website works when CSS is turned off

          The website loses its background colour, the title loses the underline, the font changes to Times New Roman, the navigation bar becomes an ordered list, the text box disappears, the images are larger, the text goes below the images, the footer loses its colour and is added as text at the bottom of the web pages, and the table in the Comments page loses its border.
